# 1. Краткое описание проекта

## 1.3. Тема проекта

Создание сцепификации и разработка информационной системы для интернет-магазина музыкального оборудования.

## 1.4. Цель проекта

Веб-приложение для интернет-магазина музыкального оборудования, способное выдерживать высокие нагрузки. Приложение должно улучшить пользовательский опыт покупателей и сделать удобным работу администраторов. В конечном итоге, разработанная система должна сделать процесс покупок в интернет магазине удобным, быстрым и понятным.

## 1.5. Задачи проекта

1. Изучить особенности и способы реализации микросервисной архитектуры;
2. Разработать архитектуру системы с учётом всех необходимых микросервисов и прочих компонентов;
3. Создать описание всех планируемых функций в системе;
4. Определить стек технологий;
5. Разработать структуру базы данных;
6. Разработать Backend WebApi;
7. Разработать Frontend.

## 1.6. Аннотация 

Современная торговля давно перешла с прилавков магазинов в интернет. Качественный сервис электронной коммерции - залог успеха малого и среднего бизнеса, предлагающего товары любых категорий. Пользователь, заходя на сайт интернет-магазина, ожидает увидеть приятный интерфейс и мгновенную работу системы. Устойчивая работа воспринимается как гарантия качества всех продуктов в целом. 

Для достижения этого необходимо тонкое инжереное понимание большого количества процессов, требующих максимальной оптимизации. Разработка масштабного приложения на базе микросервисной архитектуры требует знаний в большом количестве смежных областей. Поэтому данный проект обладает огромной академической ценностью, ведь в процессе его реализации можно освоить большое количество теоретических знаний и практических навыков, которые будут очень востребованы при дальнейшем развитии карьеры.

Разрабатываемый проект - это соврменное масштабное решение, призванное автоматизировать работу интернет-магазина. Покупатели смогут заходить на страницу клиентского приложения, выбирать товары, создавать свои корзины и оформлять заказы. Администраторы будут иметь доступ к админ-панели, которая позволит поддерживать работу магазина. В конечном итоге будет гарантироваться максимально качественный пользовательский опыт для всех.
